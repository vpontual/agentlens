not for humans

So you're a human.

You clicked the thing that said "not for humans." Bold move. This is AgentLens — a local server that converts any URL into clean, structured JSON that AI agents can actually work with. It strips the megabyte of CSS, ad-tracking JavaScript, cookie banners, and React hydration noise, then hands your agent just the content, links, and forms it needs.

It runs on localhost:7001. Your data never leaves your machine.

what your agent gets

Five response types, auto-detected

The proxy inspects each page and picks the extraction strategy that fits. Your agent doesn't need to know anything about HTML.

article
News, blogs, wikis. Clean Markdown body with metadata.
thread
Reddit, forums, Q&A. Flat-tree messages with parent IDs.
documentation
Docs sites. Sections chunked by header, code blocks and tables preserved.
serp
Google results. Structured array of titles, URLs, and snippets.
ecommerce
Product pages. Price, SKU, availability, and description.

How to point your agent here

01
Self-onboarding
Tell your agent to read the instructions endpoint. It gets a plain-text briefing with all capabilities, endpoints, and response fields.
curl http://localhost:7001/instructions
02
Tool definition
If your agent supports tool calling, give it this schema. It can then call web_browse whenever it needs to read a URL.
{ "name": "web_browse", "description": "Fetch a URL and return structured content as JSON", "input_schema": { "type": "object", "properties": { "url": { "type": "string", "description": "URL to fetch" }, "max_tokens": { "type": "integer", "default": 2000 } }, "required": ["url"] } }
03
System prompt snippet
Drop this into your agent's system prompt and it'll know what to do.
# paste into your system prompt When you need to read a web page, POST to http://localhost:7001/parse with {"url": "..."} and use the returned JSON content, links, and actions.
quick start

Up and running in 30 seconds

Docker (recommended)
docker run -p 7001:7001 agentlens # test it curl "http://localhost:7001/parse?url=https://example.com"
From source
git clone https://github.com/vpontual/agentlens && cd agentlens pip install -r requirements.txt python -m playwright install chromium uvicorn main:app --host 0.0.0.0 --port 7001
endpoints

Four endpoints, zero configuration

GET /parse?url=... Full page → structured JSON with content, links, and actions
GET /agent-manifest?url=... Links + forms only, zero content tokens — scout before you fetch
POST /batch-parse Multiple URLs, streamed as newline-delimited JSON
GET /instructions Plain-text LLM self-onboarding prompt
