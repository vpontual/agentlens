what it does
Everything your agent needs.
Nothing it doesn't.
🏃
Static-first, browser fallback
Tries a fast static fetch first. If the page is huge but extraction returns almost nothing, it knows a JS framework ate the content. Falls back to headless Chromium automatically.
render_mode: "browser"
✂️
Aggressive noise removal
Strips nav, footer, scripts, styles, and everything else designed to make your agent's day worse. Trafilatura extracts the main content; if it whiffs, markdownify takes over.
quality gate fallback
🔬
Structured extraction
Title, description, markdown content, links, forms, and interactive elements — all in one JSON response. Your agent gets a schema, not a wall of text.
agent_hint included free
🧠
Token-aware truncation
Set max_tokens and the proxy truncates cleanly at that boundary. Know exactly what you're spending before your LLM bill does.
token_estimate in response
🕵️
JS wall detection
If the HTML is large but extraction yields almost nothing, that's a JS-rendered SPA pretending to be a website. Detected automatically and escalated to headless Chromium. Domain preferences are cached.
auto-escalation cached per domain
📋
Scout before you fetch
/agent-manifest returns only links, forms, and buttons — zero content extraction. Map a site's navigation and interactive elements before committing tokens to a full parse.
GET /agent-manifest

usage
Four endpoints.
Your agent will figure it out.
GET /parse?url={url}&max_tokens={n}&force_browser={bool} full page → structured JSON
url         string                  required
max_tokens   int                    default: 2000
include_links bool                 default: true
include_actions bool                 default: true
force_browser bool                 default: false
POST /parse same, but JSON body
{ "url": "https://...", "max_tokens": 2000, "force_browser": false }
POST /batch-parse multiple URLs, streamed as NDJSON
{ "urls": ["https://...", "https://..."], "max_tokens": 2000 }
// streams results as newline-delimited JSON, 5 concurrent
GET /agent-manifest?url={url} links + forms only, zero content tokens
Returns actions, links, and agent_hint. No content extraction. Ideal for mapping before fetching.
